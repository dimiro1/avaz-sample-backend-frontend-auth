# Start Frontend (Development)

```shell
$ cd frontend
$ npm run dev
```
Frontend will be available on localhost:3000

# Start Backend (Development)

```shell
$ cd backend
$ npm run start
```

Backend will be available on localhost:3000

# Architecture

```txt
+----------+                  +----------+                +----------+                          
|  browser |  <-- http -->    | frontend |    <--http-->  | backend  |                        
+----------+                  +----------+                +----------+                        
```
